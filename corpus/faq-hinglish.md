# ZintMart Returns FAQ (English + Hinglish)

---

## Q1. Electronics return policy kya hai?

**A:** Electronics (phones, earbuds, chargers) par **7 din** ka return window hai delivery date se. Product unused hona chahiye, original box aur saare accessories ke saath. Opened hygiene items return nahi hote.

---

## Q2. Fashion / kapde kitne din mein return kar sakte hain?

**A:** Fashion items (kurtas, sarees, footwear) ke liye **15 din** ka window hai. Tags attached, unworn, bina stain ya smell ke.

---

## Q3. Refund kab aayega? Kitne din lagte hain?

**A:** QC pass hone ke baad UPI/card par usually **5–7 business days** mein refund aata hai. COD orders par wallet ya NEFT se **7–10 business days**. Exact date bank pe depend karta hai — hum invented date nahi dete.

---

## Q4. COD order return kaise hota hai?

**A:** My Orders se return raise karo. Pickup ya self-ship dono options hain. Refund ZintMart Wallet ya aapke bank account (NEFT) mein **7–10 business days** mein.

---

## Q5. Box damage mila delivery pe — kya karun?

**A:** **48 hours** ke andar photos bhejo: outer box, product, shipping label. Order ID `FZ-####` share karo. Hum priority pickup arrange karte hain agar transit damage verify ho.

---

## Q6. Galat size aa gaya — exchange ho sakta hai?

**A:** Haan, fashion par **15 din** ke andar ek **free exchange** agar size stock mein ho. My Orders → Exchange select karo.

---

## Q7. Wireless earbuds return kar sakte hain?

**A:** Haan, **7 din** ke andar agar unopened/sealed condition mein ho. Ear tips hygiene ke liye opened hone par return reject ho sakta hai — policy check karo order page par.

---

## Q8. Opened skincare return hoga?

**A:** Nahi. Beauty & personal care **non-returnable** hai hygiene reasons se.

---

## Q9. Pickup free hai ya khud bhejna padega?

**A:** Metro cities (Delhi, Mumbai, Bengaluru, etc.) mein **free doorstep pickup**. Tier-2/3 cities mein self-ship — approved seller-fault returns par **₹100 shipping credit**.

---

## Q10. Return reject ho gaya — kyun?

**A:** Common reasons: return window cross, tags missing, used/damaged by customer, non-returnable category, ya missing serial/IMEI. Email/SMS mein reason aata hai; chat par order ID se check kar sakte ho.

---

## Q11. EMI pe liya tha — refund kaise aayega?

**A:** Refund original lender ko jata hai; unke cycle ke hisaab se **10–14 business days** lag sakte hain. ZintMart side se process 5–7 din mein initiate hota hai QC ke baad.

---

## Q12. Warranty claim kaise file karun?

**A:** Return window ke baad manufacturer warranty apply hoti hai. Product page par warranty months dekho. Chat par SKU + order ID do — hum RMA steps batate hain.

---

## Q13. Gift order return — refund kisko milega?

**A:** Refund **original purchaser** ke payment method par. Gift receiver ko purchaser se coordinate karna hoga ya wallet credit option.

---

## Q14. Partial order missing hai — full return?

**A:** Missing item ke liye partial refund ya re-ship. **48 hours** mein report karo with unboxing video if possible.

---

## Q15. Invoice / GST bill chahiye return ke saath?

**A:** Invoice My Orders → Download. Return shipment mein copy optional; order ID sufficient hai.

---

## Q16. Human agent se baat kaise karun?

**A:** Chat mein likho: "human agent", "manager", ya "complaint file karni hai". Tumhe ticket ID `FZ-TKT-####` milega; agent **4 business hours** mein callback karenge.

---

## Q17. Return status kahan track karun?

**A:** My Orders → Return Details. States: Requested → Approved → Picked up → QC → Refund initiated.

---

## Q18. Defective phone — replace ya refund?

**A:** **7 din** ke andar defective electronics par replace ya full refund — restocking fee nahi. Photos + order ID required.

---

## Q19. Final Sale item return hoga?

**A:** Nahi. Checkout par **Final Sale** marked items non-returnable hain except manufacturing defect (warranty route).

---

## Q20. International card se pay kiya — refund currency?

**A:** Refund INR mein original card par; FX rate bank ka hoga. **5–10 business days** typical.

---

## Q21. Chatbot galat info de — escalate kaise?

**A:** "Speak to human" bolo ya ticket number maango. Hum policy document se cited answers dete hain; dispute par agent verify karega.

---

## Q22. Return label nahi mila email mein?

**A:** My Orders → Return → Download Label. Nahi dikhe to chat par order ID do; agent resend karega within 24 hours.
