## SKU notes for support

### ZM-EB-101 / ZM-EB-102 (Earbuds)
- Hygiene: opened ear tips may void return unless defective
- Box must include charging case + cable

### ZM-PH-201 (Nova X5)
- IMEI must match shipment record for returns
- Screen protector pre-applied — removal OK; cracks from misuse not covered in return window

### ZM-KU-501 / ZM-502 (Kurtas)
- Exchange for size within 15 days if alternate size in stock
- Custom embroidery SKUs are **Final Sale**

### ZM-SA-601 (Saree)
- Blouse piece uncut required for full refund

### ZM-FW-701 (Shoes)
- Indoor trial only; outdoor wear marks = reject
